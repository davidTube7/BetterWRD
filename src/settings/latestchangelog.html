Added

- Themes!

Go to the BWRD Settings Page and select the Change Theme button. Select a theme you like and press Save & Apply Theme or press the Enter key.

- Custom Themes!

You can use JS or CSS to style your theme. Go to the Themes page and click Create your own Theme to create themes.

- All your BWRD Settings will now save when reinstallinginfo

If you reinstall BWRD (for example: for a newer version) all of your settings, themes, your background image etc. will stay there. Unfortunately this will only work for V2.0 or above. If you just updated from a version like 1.0 or 1.5, and you're on a Chromium browser, your settings have been reset. Sorry :(. (This works by saving the settings JSON to your WRD local storage as shown in the image below).

- Sticky Navigation Bar

For those who don't know what that means, the navbar will always be at the top of your screen, even when you scroll down through the page. Enable this setting in the BWRD Settings Page.

- Refresh WRD when you change a Setting

Yeah it works, to enable, go to the BWRD Settings Page, check the last checkbox and save the settings.

- Embed strawpoll links in posts

Works for both strawpoll.me and strawpoll.com URLs. Go to the BWRD Settings Page to enable.

- Added 3 more settings for the background image feature

See what those settings do in the image below or on the background image page

- A quick loading screen for the Settings page

It shows things such as keyboard shortcuts for specific pages or fun texts at the bottom of the loading screen page, you have to be a fast reader to read those. No way to disable this yet.

Changes

- ACTUAL Custom Prefixes

Before, there was only a set of custom prefixes (REL, REQ, SCRIPT, POLL, CW, GIVEAWAY). Now, any text inside square brackets ([]) that is at the beginning of a thread title, will turn into a custom prefix.

Fixes

- Fixed some bugs

Nothing major, just minor bugs that you probably didn't even know they existed.
