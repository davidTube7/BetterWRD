Changes

Many design changes throughout the Settings pages.

Custom Theme Code now auto-saves when editing.

BWRD Settings will now be kept/saved after reinstalling the extension on Firefox browsers as well.

WRD no longer auto-refreshes on setting change if it's on a create thread/reply page.

Theme Code on Create Theme page now auto-saves and resets to default code when the Theme is created.

Dark Text Editor & Drafts Menu now work on the account manager page too (good for saving your siggies).

Drafts Menu now adapts to the current theme colors.

Dark Code Blocks & Text Editor now work only if Night Theme is on.

Clicking on a Custom Thread Prefix now redirects you to the prefix's thread.

Custom Thread Prefixes now show up on category pages as well.

Drafts Menu feature is now enabled by default and can't be turned off.

Monaco Editor Fullscreen now fills out the page, not your entire screen, allowing you to switch tabs.

Redesigned Update Popup.

Fixes

Fixed flashing of unstyled content (FOUC)

There used to be a .5 second flash on page load when BetterWRD loaded a bit later than WRD, most noticeable when using themes. Fixed by giving BWRD 0.2 seconds to fully load while the page is hidden and then show the updated page. This sadly doesn't work on Firefox for now.

Corrected/Changed design in some themes.

Fixed old layout setting not working after forum update.

Fixed a bug that caused the Dark Theme Editor on thread create page to randomly error.

Fixed a bug that caused the Drafts Menu to sometimes not show up.

Deep Night Theme.

Known Issues
